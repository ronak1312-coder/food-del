🍽️ Food Delivery Web App (Tomato Clone)

🚀 A full stack food delivery web application built using modern technologies.
Users can browse food items, add them to cart, and place orders with secure payment integration.

---

🌐 Live Demo

👉 https://food-del-ochre-seven.vercel.app

---

🛠️ Tech Stack

Frontend:

- React.js
- CSS

Backend:

- Node.js
- Express.js

Database:

- MongoDB

Payment Gateway:

- Stripe

---

✨ Features

- 🍔 Browse food items by category
- 🛒 Add to cart & remove items
- 🔐 User authentication (Login/Register)
- 💳 Secure online payment using Stripe
- 📦 Order placement & tracking
- 📱 Fully responsive design

---

⚙️ Installation & Setup

1️⃣ Clone the repository

git clone https://github.com/ronak1312-coder/food-del.git

2️⃣ Install dependencies

Frontend

cd frontend
npm install
npm run dev

Backend

cd backend
npm install
npm start

---

🔑 Environment Variables

Create a ".env" file in backend:

MONGO_URI=your_mongodb_connection
STRIPE_SECRET_KEY=your_stripe_key
